# power-todoist-card - Agent Instructions

## Project Overview

Fork of `pgorod/power-todoist-card`, a custom Lovelace card for Home Assistant to display and manage Todoist tasks.
This project is tracked in the Second Brain ecosystem to facilitate customization and contribution back to upstream.

## Available Skills

Skills are in `skills/` directory. Check for project-specific skills when working here.

## Beads Integration

This project uses Beads for issue tracking:
```bash
bd ready    # See ready tasks
bd list     # All issues
bd show <id> # Issue details
```
